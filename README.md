# Polymarket API

A production-ready Postman collection for the Polymarket API.

## Endpoints
- **Active Markets** — Get all live prediction markets
- **Resolved Markets** — Get closed/resolved markets
- **Market Categories** — Filter markets by category
- **Order Book** — Get real-time order book for any market

## Setup
1. Fork this repo
2. Open Postman → Create workspace from Git repository
3. Point it to your forked repo
4. Hit send on any request

## APIs Used
- Gamma API: `https://gamma-api.polymarket.com`
- CLOB API: `https://clob.polymarket.com`
